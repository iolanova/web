How do I play the game?

Each game starts with the selection of a theme. To do this, first select the level you want to play at: Rookie, Pro, or Expert.
At the top of the page, you will find the initial economic event. At the bottom, you will find five pre-set answer options, three of which are correct and two of which are incorrect. If you have no idea which answer might be correct, tap on the image in the middle. Contextual help will appear here.

How do I change my goal?

The default target is ten points. Each correct answer triggers a new economic event. If you correctly identify ten effects and connections, you win the game.

Can I save my scores?

Yes. Using the “Save score” button, you can decide for yourself whether your progress and results are included in the game history.
Quick note: Your results belong to you! Your personal data, progress, and successes (or failures) will not be collected or shared with third parties.

Can I delete my entire gaming history?

Yes. If you want to delete your game history, you can do so using the button at the bottom of the settings page.

Do I have to reset all the settings every time I want to play?

No. All your individual settings are saved so that you can continue playing right away the next time you start the game.




Change the exam mode

The exam mode controls the option of going back one (or more) steps in the chain of events. However, the answer options are always rearranged. If you activate this mode, this option disappears. However, this does not apply to clicking on a wrong answer. In this case, you simply start a new game.
This setting can be adjusted individually for each level. The settings are applied immediately upon returning to the game.

Change the ratio of correct and incorrect answers

Using the slider, you can adjust settings for the number of answers displayed from four to six and the number of incorrect answers from one to three.
If you reduce the number of possible answers and maximize the number of incorrect answers at the same time, the game becomes more difficult.
If you set the number of possible answers to six and reduce the number of incorrect answers to one, the probability of choosing the correct answer is highest, making the game easier.
These settings will be applied immediately when you return to the game.
